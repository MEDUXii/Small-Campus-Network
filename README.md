Small Campus Network

Small Campus Network is a simulated network infrastructure project designed to demonstrate practical campus networking concepts using Cisco Packet Tracer and Cisco IOS configurations.
The project focuses on building a secure, segmented, and scalable network for a small campus environment.

Key Features

VLAN segmentation for departments
Inter-VLAN routing
DHCP and IP addressing
Basic network security configurations
Real-world router and switch CLI setup

Technologies

Cisco Packet Tracer
VLAN, Routing, DHCP

Repository Includes

Packet Tracer topology file
Router and security configuration files
Step-by-step setup guide

Purpose

To practice real-world campus network design, routing, and security implementation.
